The JS-based Markdown Editor With All You Need Comes

SuMarkdown is designed to be easily embedded in your html page.All you need to do is to add our one js file with several dependency plugins to your page and write several lines of javascript code. Compared to Bootstrap-Markdown, we remove some useless function and provide multi-line code and code highlight, important to pragrammers.At the same time we provide a basic theme based on bootstrap which is simple but looks nice.However, it's easy for you to change its UI as your will through CSS and other tools.

What We Provide

  • a toolbar including most of your need--Bold,Italic,Link,Image,Code,Quote,Header,List and OrderList
  • hot-keys doing all what you can do through the toolbar
  • multi-line code can be present perfectly
  • code highlight based on different languages
  • enabled tab-indent in editor
  • two methods to use our editor-- Ajax load or add the theme code to your html
  • two modes--real-time effect presented and click button to preview
  • a simple theme which looks acceptable

Dependency

The Usage

Simple Way -- add elements with special class names

code example followed First,add our dependency files and our suMarkdown.js.

<link rel="stylesheet" href="lib/highlight/styles/default.css">
<script src="lib/jquery/dist/jquery.min.js"></script>
<script src="lib/highlight/highlight.pack.js"></script>
<script>hljs.initHighlightingOnLoad();</script>
<script src="lib/marked/marked.js"></script>
<script src="js/suMarkdown.js"></script>

Second, add the DOM of our editor, you can just copy the code in the /theme/basic.html file, which is recommended by us.You can also write it yourself. But please don't forget the class names that begin with su. Then we add several lines of javascript code to our html.

$(function(){
            var option={
                target:'.suMarkdown',
                preview:true,
            };
            SuMarkdown(option);
        });

The parameter of option will be explained later.

AJAX Load Way

First, the dependency files as before. Then you should add a handler for the server side to send theme file. We give an example use express.

app.get('/suMarkdown/theme/basic',function(req,res){
    res.sendFile(__dirname+'/public/suMarkdown/theme/basic.html');
});

Then add the code below to your html.

var option={
    target:'.suMarkdown',
    insert:true
};
$(function(){
    SuMarkdown(option);
});

It will append our editor to the DOM element with class-name 'suMarkdown'.

Parameter Explanation

Option Name Type Description
insert boolean true: ajax load way, false : simple way, default: false
target String,DOM Element,JQuery Object css selector grammar as String , the element it selects will wrap our editor, default:'.suMarkdown'
preview boolean true:the real-time effect mode,you will have a preview box along side to see the effect, false:the button-preview mode, you will have to click the preview button to see the effect
textHeight String Set the height of text area and preview area of our editor, you can use "500px","20%" and so on, or you can set this just through css outside
baseUrl String the url to get theme file when in ajax load mode
clean boolean true:the editor will be initialized simply in UI, false:will not, default:true, we recommend not to change it
CSS Object initialize the editor as your will
textCss Object more detaily initialize the CSS of text area of editor as your will
